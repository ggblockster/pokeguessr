## Mobile Port
This game is also available as a mobile game for Android. An iOS port might happen, but it's not easy to integrate so... Android it is.
You can install the app using Obtainium. You may download and install Obtainium from [here](https://github.com/ImranR98/Obtainium/releases) (the file is named `app-release.apk`)
1. Open Obtainium and click 'Add App'
2. Type `https://github.com/ggblockster/pokeguessr` into the 'App Source URL' section
3. Click/Tap 'Add'
4. Click/Tap 'Install'

And just like that, you should now have Pok&eacute;Guessr installed on your mobile device. Just keep in mind that you will _not_ get updates from the APK alone; Obtainium does allow for updates automatically, so that shouldn't be an issue.

Alternatively, you can also navigate to the 'Releases' section of this repository and manually download and install `app-release.apk` directly on to your device without Obtainium, but it's not recommended in the case of updates.
